Movie Trailers

To build the server­side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

REQUEST - Business Logic URL HTTP Method Request Data Format Response Data Body Format
Return a list of All movies to the user /movies GET - A JSON object holding data about all movies
Return data about a single movie by title /movies/[Title] GET - A JSON object holding data about a single movie, containing an ID, title, and some other properties. Example: 
              {
                title: 'Fight Club',
                year: 1999,
                director: 'David Fincher',
                genre: 'thriller',
                imgUrl: '',
                description: 'An insomniac office worker and a devil-may-care soapmaker form an underground fight club that evolves into something much, much more.'
              }
Return data about a director by name /movies/director/[name] GET - A JSON object holding data about a single director, containing name, birth year, death year, movies. Example: 
              {
                name: 'Luc Besson',
                bio: '',
                birthYear: '',
                deathYear: ''
              }
Allow new users to register /users POST - Created!
Allow users to update their user info /users/[name] PUT - succesfully updated.
Allow users to add a movie to their list of favorites /users/[name]/favorites POST the movie was succesfully added to the favorite list.
Allow users to remove a movie to their list of favorites /users/[name]/favorites/ DELETE - the movie was succesfully removed from favorite list.
Allow existing users to deregister (remove a user by Username or id) /users/[name]/ DELETE - Deleted
